# Лабораторная №2
## Коллекция объектов - **Агенство недвижиморсти**

# 1. Описание класса

Реализован класс `Agency`, 
который является *контейнером для объектов* недвижимости

--

## Назначение
Класс может хранить в себе объекты,
котрых можно:

- добавлять
- удалять
- сортировать
- фильровать (по критериям)
- итерировать

--

## Управвление добавлением и удалением
 - `add()` - добавляет
 - `remove()` - удаляет 
 - `remove_at()` - удаляет объект по индексу
 - `get_all()` - выводит все существующие объкты списком

--

## Методы поиска по критериям

- `find_by_price()` - ищет объекты по заданной цене
- `find_by_area()` - ищет объекты по заданной площади
- `find_by_adress()` - ищет объекты по конкретному адресу

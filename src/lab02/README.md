# Лабораторная №2
## Коллекция объектов - **Агенство недвижиморсти**

# 1. Описание класса

Реализован класс `Agency`, 
который является *контейнером* для объектов недвижимости

---

## Назначение
Класс может хранить в себе объекты,
котрых можно:

- добавлять
- удалять
- сортировать
- фильровать (по критериям)
- итерировать

---

## Валидация

- проверка на тип данных
- проверка на логические ошибки 
- проверка на минимальные значения

---

## Управвление добавлением и удалением
 - `add()` - добавляет
 - `remove()` - удаляет 
 - `remove_at()` - удаляет объект по индексу
 - `get_all()` - выводит все существующие объкты списком

---

## Методы поиска по критериям

- `find_by_price()` - ищет объекты по заданной цене
- `find_by_area()` - ищет объекты по заданной площади
- `find_by_adress()` - ищет объекты по конкретному адресу

---

## Закрытые методы для всей коллекции

- `__len__` - возвращает количесвто существующих объектов
- `__iter__` - дает возможность итерироваться по всем объектам
- `__getitem__` - возвращает объект по индексу из коллекции

---

## Сортировка

- `sort_by_price()` - сортирует объекты по цене, начиная с меньшей
- `sort_by_area()` - сортирует по площади, начиная с меньшей

---

## Фильтрация

- `get_expensive()` - фильтрация объектов для **privelege clients** с минимальным порогом цены
- `get_active()` - фильтрует только **активые** объекты
- `get_for_rent()` - фильтрует объекты только для **аренды**
- `get_for_sale()` - фильтрует объекты только для **продажи**
